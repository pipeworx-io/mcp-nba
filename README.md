# mcp-nba

NBA MCP — player, team, and game data via the BallDontLie API

Part of the [Pipeworx](https://pipeworx.io) open MCP gateway.

| Tool | Description |
|------|-------------|

## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "nba": {
      "url": "https://gateway.pipeworx.io/nba/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use nba
```
